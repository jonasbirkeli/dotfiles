# Dotfiles

This repository contains my personal configuration files (dotfiles) for macOS.

## Contents

- `.zshrc` — Zsh shell configuration
- `.zprofile` — Zsh login configuration
- `.vimrc` — Vim configuration
- `.vim/` — Vim plugins and runtime files
- `.ssh/` — SSH configuration (keys not included)
- `.vscode-oss/` — VS Code settings
